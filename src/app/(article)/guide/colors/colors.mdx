# Colors

You can use any color anywhere, all colors change with the theme.

```tsx
import { useTheme } from 'raw-ui';

const myComponent = () => {
  const { palette } = useTheme();

  return <p style={{ color: palette.accents_1 }}>myComponent</p>
};
```
