# Colors

Raw UI provides a set of colors based on the Tailwind CSS color palette by default, you can also customize it to fit your design, all colors change with the theme.

```tsx
import { useTheme } from 'raw-ui';

const myComponent = () => {
  const theme = useTheme();

  return <p style={{ color: theme.palette.neutral['600'] }}>myComponent</p>
};
```
